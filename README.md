# CFT-fixed-point-tensor-

'method' is the module defining most of the functions 

'main' is introduce the model independent functions, such as finding OPE, transformation coefficients, and correlation function

'Numerical_tensor' compute rank-3 FP tensors given the model name and conformal data

the folder 'data' includes saved ope data, transforamtion data, etc

the zip file 'cordata' include few saved correlation function. please unzip it before use

'directory' is provides path to read the corresponding correlator from cordata.
